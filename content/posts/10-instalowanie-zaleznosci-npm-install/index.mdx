NPM (z angielskiego *node package manager*) to domyślny menedżer pakietów używany w środowisku uruchomieniowym Javascript w NodeJS.
Dzięki NPM możemy instalować różne paczki, czy zależności do swojego projektu. 

Instalacja paczki jest bardzo prosta, wystarczy uruchomić komendę `npm install [nazwa_paczki]` i zacznie się ona instalować w naszym projekcie.
Od wersji NPM 5.0.0 komenda `npm install [nazwa_paczki]` zapisze zależność do sekcji `dependencies` w pliku `package.json` oraz pobierze całą paczkę wraz z zależnościami pobieranej paczki do folderu `node_modules`.
