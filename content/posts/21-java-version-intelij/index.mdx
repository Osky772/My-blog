Jak zmienić wersję Javy w Intelij Idea?

1. Z górnego panelu wybierz *File > Project Structure*. Zobaczysz taki ekran:
<Image src="./1.webp"/>
2. W zakładce *Project* pod paragrafem *Project SDK* kliknij rozwijaną listę i wybierz zainstalowaną wersję SDK 

Jeżeli nie masz jeszcze zaistalowanego SDK to w tym samym miejscu możesz je zainstalować wybierając opcję `+ Add SDK` z rozwijanej listy.
Możesz tutaj pobrać SDK lub wybrać do instalacji wcześniej ściagnięte SDK z folderu na swoim komputerze.
